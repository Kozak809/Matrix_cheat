# Matrix

### Minecraft cheat client mod

### v 0.1.0

Собран скелет мода, добавлены основные зависимости и настроен gradle.
Создан GUI

### v 0.2.0

В версии 0.2.0 была внедрена система динамического управления версиями, позволяющая централизованно изменять версию проекта в файле gradle.properties с автоматическим обновлением в HUD и метаданных мода через FabricLoader, а также добавлен полноценный модуль FullBright, который эмулирует эффект ночного зрения через систему миксинов и заменяет старую упрощенную настройку в разделе пассивных функций.

### v 0.2.1

Обновление ClickGui: настройки заменены поддержкой HEX-цветов (3 и 6 символов). Добавлена функция Screen Walk, позволяющая передвигаться в ClickGui, реализованная через новый KeyboardMixin.

### v 0.2.2

Реализован умный режим для модуля NoFall: теперь он активируется только при падении (вертикальная скорость < -0.5) и отключен при использовании элитр, что устраняет конфликты с полетом.
Добавлен модуль AutoRespawn (категория MISC), обеспечивающий мгновенное возрождение без отображения экрана смерти.
Исправлен модуль Flight: опция "Bypass Vanilla AC" теперь работает только когда игрок действительно находится в режиме полета.
