# Matrix

### Minecraft cheat client mod

---

### v 0.2.0

В версии 0.2.0 была внедрена система динамического управления версиями, позволяющая централизованно изменять версию проекта в файле gradle.properties с автоматическим обновлением в HUD и метаданных мода через FabricLoader, а также добавлен полноценный модуль FullBright, который эмулирует эффект ночного зрения через систему миксинов и заменяет старую упрощенную настройку в разделе пассивных функций.

### v 0.2.1

Обновление ClickGui: настройки заменены поддержкой HEX-цветов (3 и 6 символов). Добавлена функция Screen Walk, позволяющая передвигаться в ClickGui, реализованная через новый KeyboardMixin.

### v 0.2.2

Реализован умный режим для модуля NoFall: теперь он активируется только при падении (вертикальная скорость < -0.5) и отключен при использовании элитр, что устраняет конфликты с полетом.
Добавлен модуль AutoRespawn (категория MISC), обеспечивающий мгновенное возрождение без отображения экрана смерти.
Исправлен модуль Flight: опция "Bypass Vanilla AC" теперь работает только когда игрок действительно находится в режиме полета.

### v 0.2.3

Оптимизирована система управления: теперь в настройках Minecraft отображается только клавиша открытия меню (переименована в "Menu" в категории "Matrix"), а бинды модулей скрыты для чистоты интерфейса. 
Реализовано ручное управление вводом для скрытых биндов и улучшен ClickGUI: повторное нажатие клавиши меню теперь закрывает интерфейс.
Добавлена полноценная поддержка сохранения кастомных биндов в конфигурационном файле, что позволяет сохранять персонализированный макет интерфейса после перезагрузки.

### v 0.2.4

**Полный переход на JSON конфигурацию:**
*   Файл настроек заменен на `Matrix/matrix.json`. Теперь конфиг читаем и структурирован.
*   Реализовано корректное сохранение и загрузка состояния модулей и расположения панелей GUI.

**Исправления и улучшения:**
*   **Input System:** Полностью переписана логика обработки нажатий. Исправлен баг "фантомных клавиш", когда после смены бинда старая клавиша продолжала работать. Теперь модули (кроме Menu) используют строгую ручную проверку ввода.
*   **ClickGui:** Исправлена проблема сброса координат панелей после перезахода. Теперь расположение окон запоминается корректно.
*   **Flight:** Изменен дефолтный бинд на `F`.
